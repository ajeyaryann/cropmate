# CropMate 🌱

CropMate is an AI-powered smart farming platform that helps farmers with:

- Crop recommendation
- Weather insights
- Mandi price tracking
- Fertilizer suggestions
- Crop disease detection

## Tech Stack

Frontend
- React + Vite

Backend
- Node.js
- Express

Database
- MongoDB (coming soon)

## Project Structure

cropmate
├ backend
├ frontend
├ ai-services
├ docs
└ scripts

## Development

Backend
cd backend
npx nodemon server.js

Frontend
cd frontend
npm run dev
